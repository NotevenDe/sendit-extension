# Sendit

中文 | [English](README.En.md)

Sendit 是一款浏览器插件，可将链上图表变成实时互动直播间：在 K线 图表上原生展示弹幕，提供聊天、评论与语音房功能。

## 功能

- 单一ca下实时弹幕
- 基于ca地址的聊天与评论区
- 插件弹窗/侧边栏语音房

## 下步计划
1. 增加持仓身份验证，Holder在聊天中将会被特殊标识
2. 增加信息接口，对即时的信息消息直接通过K线通报，例如Dev买入，发推提醒
3. 增加弹幕的形式，在文字基础上增加礼物等形式（例如火箭，烟花等更多特效），并且可以与K线涨跌关联

## 用法

1. 安装插件（从源码构建可加载 `dist/` 目录）。
2. 打开支持站点内的代币页面。
3. 自动加载弹幕引擎，即可发送弹幕或者发送消息。
4. 直接点击插件菜单，可以创建/加入语音房。
